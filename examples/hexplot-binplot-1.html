Efficient Data Visualization with Hex Bin Plot

Daniel Carr from George Mason University developed a type of plot called a hexagon plot that helps address the overplotting problem in Scatter 2D plots. The plane is divided into regular hexagons (or squares), counts the number of cases in each hexagon (or square), and then (by default) maps the number of cases to the a color.
